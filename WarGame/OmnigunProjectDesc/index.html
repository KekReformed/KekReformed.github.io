Omnigun
Omnigun is a multiplayer platform fighting game where first, each player selects 3 different bullet attributes that range from the bullets element (fire, ice, rubber ect) to bullet type (orbital, snake ect). Those attributes then combine into a single gun (that gun being the omnigun) that fires each players difference bullet, the idea being to see what crazy combos players can come up with. The games development is currently on hold.
